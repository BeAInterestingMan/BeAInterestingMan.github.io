搭建基于hexo和github搭建的个人博客

1.环境

（1）安装nodejs（自行百度） nodejs  v10.20.1
（2）安装npm（自行百度）     npm     v10.20.1
（3）安装git（自行百度） 
（4）注册ithub账号（自行百度）  
（5）npm的源换一下 ，下载速度加快。cmd窗口运行命令 npm config set registry https://registry.npm.taobao.org
（6）安装hexo      cmd窗口运行命令  npm install -g hexo-cli

2.配置hexo

（1）指定一个空的文件夹，路径栏输入cmd命令，打开cmd窗口，然后运行命令 hexo init
如下表示成功，然后你会看到文件夹多了一些文件。


（2）安装hexo-git工具推送代码，在此文件夹内打开cmd 窗口 ，运行命令 npm install hexo-deployer-git –save

（2）在此文件夹内打开cmd 窗口 ，运行命令 hexo clean ， 该命令是清除命令，接着运行命令 hexo g ，该命令生成配置文件 运行命令 hexo s 该命令运行项目，然后可以看到本地博客网址，将地址localhost:4000 输入在浏览器中就可以看到初始博客页面，至此初步成功，接下来需要部署到github，让别人都可以看到你的博客。



（3）.登录github创建一个新的公开的项目


项目名称一定要是名称+.github.io,不然会报错。然后点击create 按钮创建了一个新项目

（4）在配置文件里面填写仓库地址
在项目里面找到 _congif.yml文件，找到 deploy 字段，把以下代码放进去，填写新建的仓库http地址。


退回到cmd界面 Ctrl + c 取消运行
然后输入命令 hexo d 此命令将你本地的代码push到github仓库，需要输入github用户名和密码。然后你的博客地址就是名称+.github.io



（5）个人博客搭建成功，至于模板不同是应为我换了默认主题。增加新的博客在source_posts创建.md文件,使用的是markdown语法。


（5）推荐一个好的主题 https://hexo.fluid-dev.com/docs/guide/
